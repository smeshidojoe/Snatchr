# Snatchr

### *Just Download Anything.*

---

**Snatchr** is a lightweight, fast video and audio downloader for Windows that lives entirely in your menu bar. It allows you to download media from thousands of popular websites without opening a browser tab, dealing with ads, or navigating complex command-line tools.

---

</div>

## 🏗️ Features

- **Menu bar native** — Lives quietly in your menu bar. No Dock icon, no persistent window, no interruption to your workflow.
- **Universal support** — Supports downloading from over 1,000+ popular platforms including YouTube, Twitch, TikTok, Vimeo, Facebook, Instagram, Twitter, and major news/sports outlets.
- **Batch downloading** — Paste multiple links at once. The UI automatically collapses to save space while fetching massive playlists or batches in the background.
- **Pinning & Smart Auto-Dismiss** — Quickly access the app from the system tray. Keep it always available with Pinned mode, or enable Auto-hide to automatically dismiss the window when clicking outside of it or pressing Esc.
- **Post-processing made easy** — Embed thumbnails, inject metadata, and burn subtitles directly into your downloaded files.
- **Zero-dependency setup** — Automatically configures its own fast backend engine (`yt-dlp`) locally. No manual Python or terminal setup required.
- **Premium ambient design** — Features a minimaistic, zero-overhead animated interface (choose between different themes in **About** section).

## 📥 Installation
### Recommended

Download the latest `.exe` from the [Releases](https://github.com/smeshidojoe/snatchr/releases/latest) page, open it and install to any folder you like.

#### Snatchr stores configuration files in `C:\Users\[username]\AppData\Roaming\Snatchr`
